# LyricLingo

LyricLingo is a modern music streaming web app that plays songs and shows real-time synced lyrics with live translation.

## Features

- **Music Discovery**: Search songs using JioSaavn API.
- **Synced Lyrics**: Real-time synchronized lyrics via Lrclib.
- **Translation**: Instant translation of lyric lines.
- **Favorites**: Save favorite lyrics.
- **Premium UI**: Glassmorphism and smooth animations.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack
- Next.js
- Tailwind CSS
- Framer Motion
- Zustand / Context API
